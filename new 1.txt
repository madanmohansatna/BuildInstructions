How to Build?

To initialize your local repository using the AospExtended trees, use a command like this:

  repo init -u git://github.com/AospExtended/manifest.git -b 11.x

To initialize a shallow clone, which will save even more space & time, use a command like this:

  repo init --depth=1 -u git://github.com/AospExtended/manifest.git -b 11.x

Then to sync up:

  repo sync -c -j$(nproc --all) --force-sync --no-clone-bundle --no-tags

Finally to build:

  source build/envsetup.sh
  lunch aosp_device_codename-userdebug
  m aex -j$(nproc --all) | tee log.txt

Building Spark

In case you are building Mac OS X, you are required to install coreutils from MacPorts before you continue. In order to build, use this command:

   . build/env*
   lunch spark_<devicecodename>-userdebug
   mka spark -j$(nproc --all)
